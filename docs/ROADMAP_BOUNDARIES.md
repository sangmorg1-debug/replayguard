# Public roadmap boundaries

Committed product surface: local capture/replay, deterministic evaluation, CI evidence, hosted collaboration, MCP scanning, runtime authorization, RAG provenance, cost-per-success analysis, export, and self-hosted small-team operation.

Not committed without validated demand: general-purpose observability dashboards, autonomous model changes, full enterprise IAM, every agent framework, multi-region hosted infrastructure, managed on-premises appliances, edge-fleet management, wet-lab automation, or biomedical clinical decision support.

Roadmap order is driven by retained weekly use and prevented harmful changes, not demonstration breadth.
